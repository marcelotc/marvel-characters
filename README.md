## :information_source: Para clonar o projeto:


```bash
# Clonar repositório
$ git clone https://github.com/marcelotc/marvel-characters.git

# Ir para o repositório
$ cd marvel-characters

# Instalar dependências
$ yarn install

# Rodar o projeto
$ yarn start
```
